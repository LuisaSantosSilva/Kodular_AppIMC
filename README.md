# Kodular AppIMC

O Projeto IMC feito no Kodular é uma aplicação simples em mobile feita para calcular em que categoria o usuário se encontra 

- Abaixo do Peso
- Peso Ideal
- Acima do peso
- Obesidade grau 1
- Obesidade
